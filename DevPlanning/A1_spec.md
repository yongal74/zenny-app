# Zenny 서비스 스펙 문서 (MVP 버전)

## 1. 사용자 페르소나
북미 20~35세 직장인을 타겟으로 한 3가지 페르소나를 정의. 각 페르소나는 번아웃·불안 경험자, 명상 입문자를 중심으로 실무적 행동 패턴과 니즈를 반영.

- **Alex (25세, 마케팅 코디네이터, 뉴욕)**: 주 50시간 근무 중 스트레스 쌓임. 퇴근 후 10분 명상 원하지만 지속 어려움. Tamagotchi-like 캐릭터로 재미있게 습관화 필요. 앱에서 매일 감정 체크인 + AI 코치 상담 기대.
- **Jordan (32세, 소프트웨어 엔지니어, 샌프란시스코)**: 원격근무로 고립감·불안. Zen 철학 관심 있지만 초보자라 접근성 높음. 퀘스트 미션으로 동기부여, 캐릭터 육성으로 반복 사용 유도.
- **Taylor (28세, 프리랜서 그래픽 디자이너, 시애틀)**: 불규칙 스케줄로 감정 롤러코스터. 개인화 알림·아이템 커스터마이징 원함. 인앱 결제로 프리미엄 아이템 구매하며 장기 engagement.

## 2. 핵심 가치 제안
**AI 기반 감정 케어로 매일 평온함을.**  
**Tamagotchi 스타일 Zen 캐릭터 육성으로 명상 습관 재미있게.**  
**심리학·뇌과학·철학·종교 이론으로 뒷받침된 명상 가이드.**  
**AI 생성 명상 음성 + 호흡 유도 음악으로 완전한 몰입 경험.**

## 3. 주요 기능 목록
MVP 필수 기능은 초기 런칭 핵심(가입~반복 사용 지원), 선택 기능은 post-MVP 확장으로 구분. React Native + OpenAI 통합 중심.

### MVP 필수
- **감정 일기 및 기록**: 매일 체크인 UI(이모지+텍스트 로그), 다학제 관점 기반 기분 추적. 데이터 PostgreSQL 저장.
- **AI 명상 코치 — Conversational UI (앱의 중심 화면)** EN/KO 바이링구얼:

  **상단 (~40%): 캐릭터 영역**
  - 레벨별 Zen 캐릭터 애니메이션 표시 (Lv1~7 외관 변화)
  - 터치 인터랙션 (더블탭 → 하트 파티클)
  - [Customize] 버튼: 스킨/상징/액세서리 변경 Modal

  **하단 (~60%): Conversational UI — 앱의 핵심**
  - **라운드 박스형 메시지** (cornerRadius 12px, 말풍선 아님):
    - AI: 좌측 정렬, 배경 #16203A, 흰 텍스트
    - 사용자: **1~2칸 들여쓰기** 후 시작, 배경 #6366F1, 흰 텍스트
  - **빠른 응답 버튼 (필리형)**: AI가 감정/느낌 선택지 3~6개 데이저 형태로 정렬 제시
    - 예: [행복해요] [불안해요] [피곤해요] [화나요] [슬퍼요] [모르겠어요]
    - 2열 그리드 또는 가로 스크롤 정렬, 각 버튼 높이 40px
    - 선택 시 해당 버튼 하이라이트 + 메시지로 자동 종합
  - **자유 텍스트 입력**: 버튼 선택 외 자유 타이핑 가능
  - 이론 기반 AI 응답: 심리학(CBT/MBSR), 뇌과학(DMN·세로토닌), 철학(Zen·스토이시즘·에피쿠로스), 종교(위파사나·관상기도·트란센덴탈), 호흡(4-7-8·박스·Pranayama) 서실 인용 포함
- **AI 생성 명상 음성 + 호흡 유도 음악** *(신규)*:
  - **론칭 초기 콘텐츠**: 명상 유도 보이스 **10개** + 호흡 음악 **10개** = **총 20개 사전 제작**
  - 이후 매주 2개씩 자동 추가 생성 (ElevenLabs 음성 + Suno AI / Udio 음악)
  - 각 세션 약 2분 분량 (호흡 유도 60초 + 명상 안내 60초)
  - 감정 로그 기반 개인화 (불안 → 박스호흡, 슬픔 → 위로 음악, 스트레스 → 자연음)
  - 콘텐츠 유형: 호흡 가이드 / 바디스캔 / 가이드 이미지 명상 / 자연음 + 음악
  - EN/KO 각각 제작 (EN 10개 + KO 10개 = 총 20개 론칭)
- **인터랙티브 푸시 명상 유도** *(신규)*:
  - 감정 상태·시간대 기반 스마트 푸시 (예: 오후 2시 스트레스 급등 시 "지금 2분 명상?")
  - 푸시에서 바로 호흡/명상 세션 시작 가능 (딥링크)
  - 완료 시 즉시 EXP + Zen Coins 보상 (조건반사 습관 형성)
- **Zen 캐릭터 육성 — 다중 캐릭터 시스템** **(MVP: 3종, Full: 5종)**:
  - **온보딩 시 캐릭터 선택** (1종 선택, 추가 캐릭터는 해금 가능)

  | 캐릭터 | 속성 | 성격 | 코치 말투 |
  |--------|------|------|----------|
  | **Hana** 🌸 | 꽃 정령 | 따뜻하고 감성적 | 포근하고 공감적 |
  | **Sora** ☁️ | 구름 정령 | 차분하고 지적 | 논리적·이론적 |
  | **Tora** 🦊 | 여우 정령 | 활발하고 용감 | 에너지 넘치고 격려형 |
  | *(Mizu 💧)* | 물 정령 | 유연·공감 | Post-MVP |
  | *(Kaze 🍃)* | 바람 정령 | 자유·직관 | Post-MVP |

  - 각 캐릭터 **Lv1~7 픽셀 고도화 외관** (씨앗 → 현자, 총 3×7=21장 MVP)
  - **추가 캐릭터** Zen Coins 15,000 또는 IAP로 해금
  - 장착: 스킨 20종 × 악세사리 80종 (슬롯: hat/face/body/skin/bg/pet)
- **Zen 퀘스트**: 일일/주간 미션(5분 명상, 마음챙김 로그). 완료 시 Zen Coins 보상.
- **푸시 알림**: 캐릭터 케어(배고픔/기분 저하), 인터랙티브 명상 유도. Expo Notifications 사용.
- **개인화 설정**: 더블탭 인터랙션(하트 애니메이션), 언어(EN only 초기)/알림 토글.

### 선택 (Post-MVP)
- **Zen Shop**: 80개 아이템(8카테고리: 모자/안경 등), Zen Coins 구매(RevenueCat IAP). 광고(AdMob) 보상 추가.
- **레벨 확장**: Lv8~30+ , 월간 리포트(감정 트렌드 분석).
- **소셜 공유**: 퀘스트 완료 배지 공유.
- **명상 음악 라이브러리 확장**: 주간 2개 → 누적 아카이브, 테마별 플레이리스트 (수면/집중/불안 완화).
- **음성 커스터마이징**: 코치 목소리 선택 (한국어/영어, 남성/여성), 배경음 믹서 (자연음 + 음악 비율 조절).

## 4. 유저 여정
가입부터 반복 사용까지 핵심 흐름. Figma 스타일 사용자 흐름 기반으로 마찰 최소화(초기 온보딩 3분 이내).

1. **가입 (Onboarding, 1분)**: 앱 오픈 → 이메일/Apple/Google 로그인 → 페르소나 퀴즈(3문항: 스트레스 수준, 명상 경험) → Zen 캐릭터 생성(Lv1 기본) → 첫 감정 체크인 유도.
2. **첫 사용 (Day 1, 5분)**: 홈 대시보드(캐릭터 상태+오늘 퀘스트) → 감정 로그 입력 → AI 코치 첫 대화(환영 메시지) → 간단 퀘스트 완료 → 푸시 알림 설정 요청 → Zen Coins 100개 지급.
3. **반복 사용 (Day 2+, 루프)**: 푸시로 앱 유입 → 캐릭터 케어(터치 인터랙션) → 일일 체크인+퀘스트 → AI 코치 세션 → Shop 탐색(IAP 유도) → 레벨업 시 축하 애니메이션 → 주간 리포트(선택)로 retention 강화.

| 단계 | 주요 화면 | 키 액션 | 드롭오프 방지 |
|------|----------|---------|--------------|
| 가입 | 로그인/퀴즈 | 빠른 소셜 로그인 | 프로그레스 바 + "1분만!" 카피 |
| 첫 사용 | 홈/체크인/AI 챗 | 감정 입력+퀘스트 | 가이드 툴팁 + 즉시 보상 |
| 반복 | 대시보드/Shop | 알림→퀘스트 루프 | 개인화 푸시 + 스티커 락인 |

## 5. 성공 지표
MVP 3개월 목표: DAU 1K, Retention 30%. Amplitude/PostHog로 트래킹. 북미 App Store 기준.

- **Acquisition**: DL 수(10K), CAC <$2 (AdMob+ASO).
- **Activation**: D1 Retention 50%, 첫 체크인 완료율 70%.
- **Engagement**: DAU/MAU 25%, 세션당 시간 5분+, 퀘스트 완료율 40%.
- **Retention**: D7 30%, D30 15%. 코호트 분석(푸시 온/오프 비교).
- **Conversion/Revenue**: IAP 전환율 5% (Zen Coins $4.99 팩), LTV $10 (3개월). AdMob eCPM $5+.
- **Churn 이유**: 로그아웃 사유 설문(번아웃 감소 미측정 등). A/B 테스트: 알림 빈도 최적화.
